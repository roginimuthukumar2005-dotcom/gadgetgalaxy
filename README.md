# Gadget Galaxy

Demo modern e-commerce frontend built with HTML, CSS and JavaScript.

Structure:

- `index.html` — Home
- `products.html` — Products list with filters
- `product.html` — Product details
- `cart.html` — Cart
- `wishlist.html` — Wishlist
- `about.html`, `contact.html`, `login.html`, `signup.html`
- `css/style.css` — Styles
- `js/main.js` — Product data and UI logic (localStorage-based cart/wishlist)

To run: open `index.html` in a browser (no server required). Pages link together and use `localStorage` for cart/wishlist.
